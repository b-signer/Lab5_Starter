### Unit test questions
1. I would not use a unit test for a "message" feature,
since this is too extensive for a single unit test, and would
need numerous tests testing different parts of the process,
so one could know where things went wrong.
2. I would use a unit test to test the "max message length"
feature, since it is a very specific and narrow thing to test
that can be encapsulated in a single unit test.
